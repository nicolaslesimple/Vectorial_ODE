Detailed Description

Class implementing the Runge-Kutta 2 steps to solve the ODE system. Runge-Kutta method allows the solving of the ODE system. This class is implementing in such a way that it only writes the solution every WriteOutputTimestep. This class is inherited from RungeKutta class also inherited from VectorialODE.
