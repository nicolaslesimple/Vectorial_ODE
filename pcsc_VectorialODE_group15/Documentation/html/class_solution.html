Detailed Description

This class allows the saving in the memory and the writing in a output file of the solution of the ODE system. The constructor need the input object defining the system to create a consistent format for the solution. In this class, method allowing the retrieve, the modification and the writing of the solution are implemented. The attributes of this class are the the solution matrix and two integer defining the size of this solution.
