 CInputClass allowing the saving of all the variables to define the system we want to solve. Moreover, Input object needs 3 matrix to define the system : InitialConditionMatrix, CoefficientMatrix, FunctionMatrix. Matrix are defined to be vector of vector of double. Input variables are given to the input class by the constructor. The other methods are getter allowing the access to theses private attributes
